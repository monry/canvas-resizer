# What?

uGUI の Canvas を良い感じにリサイズします。

# Why?

* スマホの画面サイズに応じて背景を調整したかった。

# Install

```shell
$ npm install @umm/canvas-resizer
```

# Usage

* Canvas がセットされている GameObject に CanvasResizer を AddComponent してください。
* 後はヨシナに CanvasScaler の matchWidthOrHeight を調整します。
